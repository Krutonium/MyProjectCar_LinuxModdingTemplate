# @MOD_NAME

A mod for My Summer/Winter Car using the MSCLoader framework.

## Description

@MOD_NAME is a mod that enhances your gameplay experience in My Summer/Winter Car. This mod is built using the MSCLoader
framework and provides additional functionality to the game.

## Features

- Built with MSCLoader framework
- Compatible with My Summer/Winter Car
- Easy to install and configure

## Requirements

- MSCLoader framework installed
- My Summer/Winter Car game

## Building the Mod

To compile the mod, use one of the following commands:

**Release Build:**  
`dotnet build -p:Configuration=Release`  
**Debug Build**  
`dotnet build -p:Configuration=Debug`
